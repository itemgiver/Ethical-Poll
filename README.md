### 개발 사전 준비 사항

##### node

```bash
node v14.18.1
```

##### yarn

```bash
yarn 1.22.1
```

### 개발 규칙

commit 전에

1. `yarn format`
2. `yarn check:tsc`

### 개발 방법

1. node_modules 설치 `yarn install`
2. 개발서버 실행 `yarn dev`
3. 브라우저로 3000포트 접속 http://localhost:3000
